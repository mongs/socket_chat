# socket_chat
使用node, express, socket.io
制作一个简单的实时聊天室

# 使用方式

## clone 文件
```
$ git clone https://github.com/mongs/socket_chat.git
```

## 进入socket_chat目录
```
$ cd socket_chat
```

## 安装node模块
```
$ cnpm install express socket.io
```

## 运行项目
```
$ node app
```

## 浏览器访问[localhost:3000](localhost:3000)
可以多开几个页面,相互聊天
